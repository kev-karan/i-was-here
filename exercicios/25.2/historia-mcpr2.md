# O dia que eu quase fui de base programando em java

## Sobre o que é esta história?
Minha experiência com a linguagem de programação em Java

## Relato
Era um lindo dia no ano de 2023, ensolarado e parecia que o mundo inteiro estava feliz, menos eu, uma estudante de ciências da computação sendo obrigada a realizar cálculos de matriz na linguagem de java.

## Por que isso é relevante para alguém da área de computação?
Poque eu acho que todo estudante se sente assim uma vez na vida.
