# Super pato direto do ovo

   ## Sobre o que é esta história?
   Meu primeiro projeto de animação 3d feito durante o ensino médio.

   ## Relato
   Durante meu ensino médio, eu fiz um curso técnico de programação de jogos digitais, e, durante esse tempo, meu professor pediu para fazermos grupos e criarmos uma animação 3d usando blender. Foi uma situação desafiadora. Entretanto, foi um acontecimento que me fez perceber como a área de tecnologia pode ser diversa, não apenas em criação de linhas de código. Para quem queira ver esse projeto que me trouxe para a área de tecnologia, aqui está o link: [SuperPato: Direto do ovo](https://youtu.be/EWD89xHa2Pg?si=O_XrShoL90138ZWL)

   ## Por que isso é relevante para alguém da área de computação?
   Essa história é relevante pois mostra como o primeiro contato com tecnologias novas pode despertar o enteresse em um aluno para novos pontos da tecnologia.
